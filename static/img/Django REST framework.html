University Instance

Retrieves a list of all Universities

1. Search Values

Search by university by name keyword

  • Example:

    • Search by University Name: ?search=Universidad de Puerto Rico

2. Ordering Values

Order universities by id, name or city

  • Examples:

    • Ordering by id: ?ordering=id
    • Ordering by name: ?ordering=name
    • Ordering by city: ?ordering=city

The API may also specify reverse orderings by prefixing the field name with '-', like so:

  • Reverse ordering: ?ordering=-name

Multiple orderings may also be specified:

  • Multiple ordering: ?ordering=name,city

3. Filters Values

Filters by name or city.

  • Examples:

    • Filter by name: ?name=Universidad de Puerto Rico
    • Filter by city: ?city=San Juan
